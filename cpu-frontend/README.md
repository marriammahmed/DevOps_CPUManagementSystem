# CPU Management App - Frontend

React application for managing CPUs and sockets.

## Prerequisites
- Node.js and npm installed
- Backend running on `http://localhost:8080`

## Installation
```bash
npm install
```

## Running the App
```bash
npm start
```

Opens at `http://localhost:3000`

## Features

- View list of all CPUs
- Add new CPUs
- Edit existing CPUs
- Delete CPUs
- Select sockets from dropdown
- Success animation with confetti

## Backend Repository

https://github.com/marriammahmed/cpu-app

## How to Use

1. **View CPUs:** Table displays all CPUs with brand, model, and socket
2. **Add CPU:** Click "+ Add CPU" button, fill form, click Save
3. **Edit CPU:** Click "Edit" on any CPU, modify data, click Save
4. **Delete CPU:** Click "Delete" on any CPU (requires confirmation)

## Technologies

- React
- Fetch API
- CSS Styling
